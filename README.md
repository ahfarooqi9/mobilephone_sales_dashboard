## 📌 Project Overview

This project showcases a **dynamic and interactive Power BI dashboard** designed to analyze mobile sales performance across multiple dimensions including geography, time, customer ratings, and payment methods.

The dashboard helps in identifying key trends, understanding customer behavior, and supporting data-driven decision-making.

---

## 🎯 Key Insights

* 📍 **City-Level Sales Analysis**
  Identifies top-performing cities contributing to overall revenue

* 📈 **Monthly Sales Trends**
  Tracks fluctuations in total quantity sold across months

* 📅 **Day-Wise Performance**
  Highlights sales variation throughout the week

* 📊 **Brand & Model Analysis**
  Compares performance across different mobile brands and models

* 💳 **Payment Method Distribution**
  Shows dominance of UPI, cards, and cash transactions

* ⭐ **Customer Ratings**
  Provides insights into customer satisfaction levels
